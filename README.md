# WhatsApp-Bot
Este é um bot para WhatsApp desenvolvido com a biblioteca WhatsApp-web.js. Ele permite que você interaja com o WhatsApp de forma programática.

Funcionalidades
- Responde a mensagens de texto com opções pré-definidas
- Envia mensagens de resposta personalizadas com base nas opções selecionadas

Instalação
1. Clone o repositório: git clone https://github.com/seu-usuario/seu-repositorio.git
2. Instale as dependências: npm install
3. Configure o arquivo auth_info.json com suas credenciais do WhatsApp
4. Inicie o bot: node index.js

Uso
1. Envie uma mensagem de texto para o bot com a palavra "ping"
2. Selecione uma opção pré-definida (A, B, C, etc.)
3. O bot enviará uma mensagem de resposta personalizada com base na opção selecionada
